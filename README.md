# Skill 3 – Hibernate HQL Lab

This project demonstrates the use of Hibernate Query Language (HQL) in Java using Hibernate ORM.

## Features
- Hibernate configuration with MySQL
- Entity mapping using Product class
- HQL queries for:
  - Fetching records
  - Filtering data
  - Updating records
  - Deleting records

## Technologies Used
- Java
- Hibernate ORM
- MySQL
- Maven
- Spring Tools Suite (STS)
